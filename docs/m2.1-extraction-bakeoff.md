The replay fixture is intentionally deterministic and exists so default CI can test
schema validation, matching logic, cost calculation, and reporting without calling
DeepSeek.

The fact-level gold file is an exhaustive draft label set. The benchmark now reports
a scorecard instead of legacy tuple precision/recall:

- `grounded_precision`: claims grounded in the same chunk text divided by total claims.
- `coverage_recall`: chunk-scoped matched gold facts divided by all gold facts.
- `primary_coverage_recall`: the same recall restricted to primary facts.
- `numeric_grounding_rate`, `direction_accuracy`, `magnitude_accuracy`, and
  `subject_accuracy`: separate diagnostic axes.
- `redundancy_rate`: grounded claims that did not cover a distinct fact.
